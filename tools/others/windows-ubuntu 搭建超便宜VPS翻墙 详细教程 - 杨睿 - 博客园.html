根据别人的推荐，笔者使用代理服务Shadowsocks，其优点是稳定、跨平台、而且在搬瓦工的自带操作系统上可以一键安装，免去了很多麻烦。我们点击左边选项框最后一项进行安装。

安装好了之后，即可得到shadowsocks的各项参数，如下图所示：

其中，第一行为加密方式，可以不用管他，记住就好；第二行为服务器端口，注意这个VPS的SSH端口不一样哦；第三行为服务密码。当你在上一步点击安装shadowsocks之后，它事实上不仅仅帮你安装了shadowsocks，而且还利用ssserver命令对电脑进行了服务端配置，并且该配置是开机启动的，配置参数的一部分即是上图的三个值。

Tips

shadowsocks的配置包括两块：

  • 服务端，对应指令 ssserver， 布置在我们购买的VPS上
  • 客户端，对应指令 sslocal， 布置在我们需要FQ上网的，自己的电脑上

以上搬瓦工帮我们完成了服务端的配置，但是客户端则需要我们自己配置。

第三步 客户端的配置

windows

windows 只需要按照搬瓦工的shadowsocks自带的介绍下载其客户端软件即可，非常方便，如图所示

linux

笔者自己的系统为ubuntu16.04，其配置起来十分的方便，首先我们需要安装shadowsocks客户端：

# 安装包，它是由python写的
sudo pip install shadowsocks

#创建文件
gedit /home/ray/shadow.json

在json文件中写上配置信息：

{
"server":"*.*.*.*",
"server_port":65432,
"local_port":1081,
"password":"****",
"timeout":600,
"method":"aes-256-cfb"
}

其中server是服务器id，这里需要读者自行填入，即我们购买的VPS的ip地址，"server_port"是shadowsocks的端口，在上图的第二行中得到，然后"local_port"是本地端口，注意：注意： 该端口号不能和本机已使用的端口号冲突，否则就会抛出“[Errno 98] Address Already in use”的错误。我们可以使用

sudo netstat -tulpn

查看所有端口号，然后选择合适的端口填入，最后的密码和加密方法"method"按照之前图中shadowsocks的值填入即可。

完成以上文件之后，我们只需要调用命令

sslocal -c /home/ray/shadow.json

即可，成功后显示如图所示

后续工作2 ubuntu开机启动

上述过程每次都得打开新的终端输入sslocal非常不方便，我们可以让其开机自动启动，步骤如下：

  1. 新建sh文件shadow.sh，内容为

    #！/bin/bash
#shadow.sh
sslocal -c /home/ray/shadow.json

  2. 找到计算机中的rc.local文件，一般位于分区/etc下，找到后进行修改：

sudo emacs /etc/rc.local
# 在倒数第二行加入如下命令并保存
sh /home/ray/shodow.sh

重启体验一下完全无墙的感觉，真的是爽到爆了！
