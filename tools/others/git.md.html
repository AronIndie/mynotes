# 分支操作
1. 创建并切换分支
git branch dev     # 创建分支dev
git checkout dev   # 切换到分支dev
上述两个命令等价于git checkout -b dev
2. 查看当前分支
git branch
3. 用当前分支提交
git add readme.txt 
git commit -m "branch test"
    [dev fec145a] branch test
    1 file changed, 1 insertion(+)
4. 我们把dev分支的工作成果合并到master分支上：
git checkout master
git merge dev
5. 合并后即可删除dev分支
git branch -d dev
