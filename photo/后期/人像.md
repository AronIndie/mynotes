
# 磨皮


## 通道磨

1. 选对比度最高的通道，duplicate一份

2. Filter - other - highPass

3. image - calculation: blending : hard light 或者 叠加，确定

4. 上述步骤重复三次

5. 选区，反选

6. 曲线 点中间，往上提
